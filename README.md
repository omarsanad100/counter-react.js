# Counter Component

This is a simple React project that implements a counter component. The counter allows the user to increase and decrease the count value. The project also uses CSS modules for styling.

## Features

- **Increase Count**: The count value can be increased by clicking the "increase" button.
- **Decrease Count**: The count value can be decreased by clicking the "decrease" button. If the count reaches 0, a prompt message appears.
- **CSS Modules**: The component is styled using CSS modules to ensure scoped and modular styles.
